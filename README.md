# My Portfolio

This is my personal portfolio website built to showcase my projects, skills, and tech expertise. It is fully **responsive**, visually engaging, and interactive, making use of modern frontend technologies to create a smooth user experience.

## Features

- Interactive UI with **animations and transitions** using Framer Motion and Lottie
- Typing animations with **Typewriter Effect**
- Contact form powered by **Formspree**
- Fully **responsive device mockups** with HTML5 Device Mockups
- Real-time notifications with **React Hot Toast**
- Analytics integration via **React GA4**
- Smooth scrolling effects

## Technologies Used

- **React 19** – Core UI library
- **Next.js 16** – Server-side rendering and optimized routing
- **Styled Components 6** – Component-level styling
- **Framer Motion** – Animations and transitions
- **Lottie React** – Rendering animated illustrations
- **Typewriter Effect** – Typing animations
- **React Hot Toast** – Toast notifications
- **React GA4** – Google Analytics tracking
- **HTML5 Device Mockups** – Responsive device previews
- **TypeScript 5** – Type safety and enhanced developer experience
- **ESLint** – Code linting and standards
